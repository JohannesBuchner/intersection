Line-Intersection formulae

Ray tracing formulas for various 2d and 3d objects were derived using the computer-algebra system sympy.

The collection currently contains:

  • 2d_line - ellipse intersection
  • 2d_line - circle intersection
  • line - hyperbolical_paraboloid intersection
  • line - hyperboloid intersection
  • line - cone intersection
  • line - plane intersection
  • line - elliptical_paraboloid intersection
  • line - sphere intersection
  • line - ellipsoid intersection
  • line - hyperboloid_twosheets intersection
  • line - cylinder intersection
